Algorithm Info

Complexity: O(n log n)
Approach: Sorted Polar Angle
Graham scan uses a stack to find the convex hull of a finite set of points in the plane with a time complexity of O(n log n).

Algorithm Logic

Graham Scan (O(n log n)):

Find point P with lowest Y (base)
Sort remaining points by polar angle with P
Initialize Stack with first 3 points
FOR each point P[i] from sorted list:
  WHILE angle of (next-to-top, top, P[i]) is not left turn:
    Pop point from stack
  Push P[i] to stack
RETURN stack as Convex Hull

Jarvis March (O(nh)):

leftmost ← point with minimum X-coordinate
p ← leftmost
do:
  add p to hull
  q ← (p+1) mod n
  for each point r in points:
    if orientation(p, q, r) == counterclockwise:
      q ← r
  p ← q
until p == leftmost
